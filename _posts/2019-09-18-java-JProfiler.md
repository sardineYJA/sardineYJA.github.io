# JProiler

JProfiler监控软件下载安装：https://www.ej-technologies.com/download/jprofiler/files

IDEA 插件下载：File --> Settings --> Plugins --> 搜索 JProfiler 并安装 

IDEA 配置：File --> Settings --> Tools --> JProfiler --> jprofiler.exe目录
